# 隱私權政策: Chromium Bridge

> 本文是 [privacy-policy.md](./privacy-policy.md) 的繁體中文翻譯。以英文版
> 為準; 简体中文版見 [privacy-policy.zh_CN.md](./privacy-policy.zh_CN.md)。

_最後更新: 2026-07-17_

Chromium Bridge 是一個開源瀏覽器擴充功能, 透過一個執行在你自己電腦上的原
生訊息主機, 把 MCP 用戶端 (如 Claude Code、Claude Desktop 或 Codex) 連接到
你本機的 Chromium 瀏覽器。本政策說明擴充功能會存取什麼, 以及它對這些資料
做什麼、不做什麼。

## 摘要

**Chromium Bridge 不收集、不傳輸、不出售任何個人資料。** 它沒有分析統計、
沒有遙測、沒有遠端伺服器。擴充功能做的一切都發生在你自己的機器上: 網站只
有在你核准之後才能被操作, 每個高風險操作都會請求你的確認。

## 擴充功能能存取什麼

為了讓經你核准的 AI 代理操作你已登入的頁面, 擴充功能可以 (**僅限你明確核
准的網站**; 或在你明確開啟 `allowAllSites` 後, 所有網站):

- 讀取目前頁面的內容 (DOM、文字、表單欄位)。
- 讀取作用中網站的 Cookie, 包括 `httpOnly` Cookie。
- 讀取 Web 儲存 (`localStorage` / `sessionStorage`)。
- 在頁面中執行 JavaScript。
- 把本機檔案附加到頁面的檔案輸入框 (預設關閉; 每次使用都顯示確切路徑供確
  認)。

與網站核准無關, 它還可以讀取開啟的分頁清單 (標題和 URL), 以及開啟、切換
或關閉分頁; 關閉分頁會請求確認。

含憑證的值 (Cookie 和 Web 儲存) 是**唯讀的** (擴充功能在設計上沒有寫入或
修改 Cookie 或儲存的 API), 並在回傳前**遮罩** (JWT、長十六進位字串、長數
字串會被抹除)。

## 資料如何使用、去向哪裡

- 所有通訊都留在**你自己的電腦上**。擴充功能經 Chrome 的原生訊息通道與本
  機原生訊息主機通訊; 該主機經一條私有的、經過身分驗證的本機通訊端與你的
  MCP 用戶端通訊。
- **擴充功能不向其作者或任何自有伺服器傳送任何資料。** 它自身不發起任何對
  外網路請求。代理讀取的內容會在同一台機器上、應你的請求回傳給你設定的
  MCP 用戶端; 該用戶端接下來如何處理這些資料 (例如傳送給它使用的 AI 服
  務), 由該用戶端自己的隱私權政策約束, 而不在本政策範圍內。

## 同意與控制

- **逐網站核准。** 在提示中核准某個來源之前, 該網站的頁面無法被讀取或操
  作。
- **逐操作確認。** 高風險操作 (表單提交、按鍵、關閉分頁、檔案上傳, 以及每
  一次 JavaScript 執行) 都會在一個網頁無法看見、無法互動的擴充功能自有視
  窗中請求確認。在已完成 Touch ID 註冊的 Mac 上, 最高風險的操作需要
  Touch ID 核准。這些確認預設開啟; 每一項都是由你控制的設定。
- **緊急停止開關。** 你可以隨時從擴充功能的選項頁、桌面應用程式或命令列停
  止橋的一切活動; 解除需要你本人在場的明確核准。

## 擴充功能在本機儲存什麼

擴充功能只在你的裝置上、瀏覽器的本機擴充功能儲存中保存少量設定:

- 你核准的網站清單 (允許清單)。
- 你的擴充功能設定/偏好。
- 如果你完成了註冊: 用於驗證你自己電腦安全硬體的公鑰指紋 (絕不是私鑰: 私
  鑰留在 Secure Enclave 裡)。
- 一份有界的、本機的近期安全決策日誌 (確認、撤銷), 可在選項頁檢視。

這些資料絕不離開你的裝置, 並在你移除擴充功能時被刪除。

## 遠端程式碼

擴充功能**不**載入或執行遠端託管的程式碼。可能在頁面中被執行的
JavaScript 是你 (或在你指示下的 MCP 用戶端) 在本機提供的程式碼, 絕不會從
遠端來源取得。

## 資料共享與出售

Chromium Bridge **不**向任何人出售或共享你的資料。唯一會收到資料的一方,
是你自己設定並接上這座橋的 MCP 用戶端。
